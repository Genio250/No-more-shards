A simple mod that sets your mask shards and vessel fragments to 3 every time they are empty, essencially making so every shard or fragment will grant a complete mask or vessel.

WARNING: If you open any file your mask shards and vessel fragments will be set to 3. As this is my first mod, I was unable to sync this to the process of getting mask shards or vessel fragments itself.
If you accidentally open an old file you can force close the game with AltF4 or turning off your device, otherwise you will have to save edit your shards back.
